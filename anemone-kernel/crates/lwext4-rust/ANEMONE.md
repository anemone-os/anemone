# lwext4_rust in Anemone

This crate is maintained in-tree for Anemone's ext4 filesystem support.

Imported sources:

- `lwext4_rust`: `502b48b33b5a6b6a75f78edc6c979c2638fc9323`
- `c/lwext4`: `430e426a0e98959c8dd05c1839ecc55ff1e4938c`

The upstream repository metadata was intentionally not imported. Future changes to the Rust wrapper or the bundled C `lwext4` sources should be made directly in this directory and reviewed as normal Anemone changes.
