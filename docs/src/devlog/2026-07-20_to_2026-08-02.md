## 2026-07-23 - LoongArch / 2K1000 / AHCI 与运行期诊断提交回填

**Date:** 2026-07-23
**Authors:** EDGW, Codex
**Area:** LoongArch / AHCI / console / KUnit / task lifetime / TID / ring buffer

**Summary:** 本日提交完成 LoongArch trap 对齐、generic AHCI controller、canonical stdin、IPI/延迟
销毁诊断、单核 KUnit 适配、`Arc<Task>` 生命周期修复、默认关闭 TID reuse、2K1000 build/config
接入和 ring buffer 尾部弹出；AHCI 后续又完成了从 block module 到 driver 根模块的行为保持型移动。

**Change:**

| Commit | 事实摘要 |
| --- | --- |
| `079c13d9` | 将 LoongArch kernel/user trap entry 对齐到 4 KiB page base，并清理 user trap 错误日志尾部。 |
| `48f86615` | 增加 generic AHCI 1.x 单 port、slot-zero polling、DMA bounce、ATA IDENTIFY、LBA48 DMA EXT block driver。 |
| `e0ae2ac5` | 为 console/ns16550a 增加固定容量 canonical stdin、回显、退格和非阻塞/信号感知读取。 |

| `a73ab58e` | deferred task disposal 在队列压力升高时记录队列长度和 task identity。 |
| `c881bcef` | 单 CPU 配置下收窄 procfs/affinity/scheduler KUnit 假设，避免引用不存在的 CPU。 |
